# Corre-Cabrito-Game
"Corre Cabrito" é um jogo de tabuleiro digital em Java (Swing). O Cabrito (C) tenta evitar a captura pelo Carcará (A) em um tabuleiro de 6 nós. A lógica implementa regras de adjacência, captura pelo Carcará e um Super Pulo exclusivo e único para o Cabrito. O projeto inclui controle de volume e menus de jogo.
